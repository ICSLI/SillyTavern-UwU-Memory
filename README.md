# SillyTavern UwU Memory Plugin

Server-side plugin for [SillyTavern-UwU-Memory](https://github.com/ICSLI/SillyTavern-UwU-Memory) extension.

This plugin provides LanceDB vector database support for semantic memory search.

## Installation

1. Copy this folder to `SillyTavern/plugins/uwu-memory/`
2. Run `npm install` in the plugin folder
3. Restart SillyTavern

## Requirements

- SillyTavern with plugin support enabled
- [SillyTavern-UwU-Memory](https://github.com/ICSLI/SillyTavern-UwU-Memory) extension (main branch)
